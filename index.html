Image hover effects can help to make a website have a more interactive, dynamic feel, and if done right, just look nice. Some examples include making the image move, grow, shrink, rotate, or change color. Here, we'll be using the :hover pseudo-class to determine what happens when the user hovers over an image. A pseudo-class, if you remember, defines a special state of an element. We'll stick to images here, but the :hover psuedo-class can also be useful for other things, like links. To actually make the effect happen, we can use the transform property. To change the size of the image, we can use transform: scale(). For instance, transform: scale(1.5) will increase the image's size by 50%, while transform: scale(0.5) will decrease it by 50%. On the other hand, transform: rotate() will rotate the image. For example, transform: rotate(90deg) will rotate the image by 90 degrees. By using transition: transform we can specify how long it take for the change to take effect. For example, here we use transition: transform .3s.
